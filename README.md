# nogglee.github.io

이 레포지토리는 **개인 포트폴리오 웹사이트의 소스 코드**입니다.  

- **배포 URL**: https://nogglee.github.io  
- **핵심 포인트**  
  - HTML/CSS 기반 정적 웹 페이지 직접 구현  
  - Git을 활용한 코드 버전 관리(add / commit / push)  
  - GitHub Pages를 통한 배포 경험  

본 프로젝트는 프레임워크 없이 웹의 기본 구조와 스타일링 역량을 보여주는 데 목적이 있습니다.
